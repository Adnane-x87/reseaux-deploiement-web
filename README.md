# So... What Is Ubuntu?

Ubuntu is a free and open-source operating system based on Linux.

- Created in 2004 by Canonical Ltd.
- Built for personal computers, servers, cloud platforms, and more
- Used by students, developers, scientists, businesses, and schools worldwide

The name "Ubuntu" reflects ideas of sharing, community, and helping each other.

---

# Wait... It Is Free?

Yes. Ubuntu is 100% free to download and use.

| System | Cost |
| --- | --- |
| Ubuntu | Free |
| Windows | Paid, usually around $100-$200 |

Ubuntu is especially attractive for:

- Students
- Developers
- Companies
- Schools and public organizations

---

# What Does Ubuntu Look Like?

Ubuntu uses a clean, modern desktop called GNOME.

It includes:

- A file manager
- A web browser
- Text editors and office tools
- Music and video players
- An App Center for installing software

It feels different from Windows at first, but it is easy to learn.

---

# What Does "Open Source" Mean?

Open source means the source code is available for anyone to read, modify, and share.

Think of it like a recipe:

- Closed source: you can use the final product, but the recipe stays secret
- Open source: you can study the recipe, improve it, and share it with others

That openness helps a global community improve Ubuntu continuously.

---

# Is Ubuntu Safe?

Yes. Ubuntu is known for strong security.

- Password protection is required for system-level changes
- Malware is much less common on Linux-based systems
- Security updates are released regularly
- Open code means bugs can be found and fixed quickly

---

# Where Is Ubuntu Used?

Ubuntu powers many systems people use every day.

- Websites and servers
- Cloud computing platforms
- Science and research labs
- Robotics and AI projects
- Personal computers
- Schools and universities

Examples often associated with Linux and Ubuntu environments include major web platforms, AWS, Google Cloud, Azure, NASA, CERN, and academic institutions.

---

# Ubuntu Releases

Ubuntu has two main release rhythms:

- Every 6 months: a new regular release
- Every 2 years: an LTS release with 5 years of support

Examples:

- Ubuntu 24.10 is a regular release
- Ubuntu 24.04 LTS is a long-term support release

For beginners, the best choice is usually the latest LTS version.

# How To Try Ubuntu

You do not need to replace Windows to explore Ubuntu.

1. Live USB - boot Ubuntu from a USB stick without installing it.
2. Virtual Machine - run Ubuntu inside Windows with software like VirtualBox.
3. Dual Boot - install Ubuntu alongside Windows and choose at startup.

---

# Useful Commands for Beginners

The terminal is a text-based control panel for your system.

```bash
# Update your system
sudo apt update && sudo apt upgrade

# Install a program
sudo apt install vlc

# Show the current folder
pwd

# List files
ls

# Clear the screen
clear
```

The terminal can look intimidating at first, but it becomes very useful quickly.
