🔓 What Does "Open Source" Mean?

Think of it like a recipe 🍕

🍕🔒

Closed Source

The recipe is secret.
You can only eat the pizza.
(Like Windows)

🍕🔓

Open Source

The recipe is public.
You can cook it, improve it, and share it!
(Like Ubuntu)

A huge global community constantly improves Ubuntu for free!

⌨️ Useful Commands for Beginners

The Terminal is like a text-based control panel

Terminal
user@ubuntu:~$ sudo apt update && sudo apt upgrade
# Update your system
Reading package lists... Done
user@ubuntu:~$ sudo apt install vlc
# Install a program (example: VLC)
Installing vlc...
user@ubuntu:~$ pwd
# See your current folder
/home/user
user@ubuntu:~$ ls
# List files in a folder
Documents Downloads Music Pictures
user@ubuntu:~$ clear
# Clear the terminal screen

💪 Don't be scared of the terminal — it's very powerful and you'll love it!
